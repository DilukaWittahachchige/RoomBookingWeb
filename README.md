# RoomBookingWeb
Room booking manage web application

Agile Project management with TDD

This project consists of two parts 
   .net core 5 Rest Web API- backend 
   Angular   12 - Frontend 


Backend Project ----------------------------------------------------------

n- tier architecture

Database layer 
  MS SQL server 
  
DataAccess Layer
 Repository pattern, 
 Unity Of Work, 
 Entity Framework core 
 
Common Layer
 Dependency injection 
 Cache - Inmemory and Redis
 Domain data classes 
 Logger 
 
Business Logic Layer 
  Business services 
  
Presentation Layer 
  Web Api (Rest)
  
Tests
  xUnit with Rhino Mocks  

Scheduler 
 Send email and update room table room status 

Frontend Project ----------------------------------------------------------
  Angular client application (Version 12)
